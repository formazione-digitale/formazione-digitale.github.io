Cos'è una sottorete?

Una sottorete (subnet) è una partizione logica di una rete IP. Dividere una rete grande in reti più piccole permette di ridurre il traffico di broadcast e migliorare la sicurezza isolando i segmenti.

// Perché dividere le reti:
  • Riduzione congestione: limita i domini di broadcast.
  • Ottimizzazione IP: evita sprechi di indirizzi in reti troppo ampie.
  • Sicurezza granulare: permette di isolare i segmenti critici (es. server vs guest).
2

L'operazione di AND Logico

Il computer determina l'appartenenza di un IP a una rete eseguendo un AND binario tra l'IP host e la Subnet Mask. È l'operazione booleana fondamentale per il routing.

La regola: Il bit risultante è 1 solo se entrambi i bit di input sono 1. Se la maschera presenta uno 0, il bit host dell'indirizzo IP viene "azzerato" nel calcolo della rete.
3

Indirizzi riservati

In ogni sottorete, due indirizzi sono riservati e non possono essere assegnati a PC, server o altri terminali host:

  • Network ID: Identifica la rete. Tutti i bit host (rossi) sono a 0.
  • Broadcast Address: Usato per comunicazioni "uno-a-tutti". Tutti i bit host (rossi) sono a 1.
Formula Host: Il numero di host utilizzabili è calcolato come 2n - 2, dove n è il numero di bit host (rossi).
